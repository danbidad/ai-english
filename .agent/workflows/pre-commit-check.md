---
description: 코드 작업 완료 전 품질 및 규칙 준수 여부 확인
---

모든 코딩 작업 후, 파일 저장을 확정하기 전에 다음 절차를 반드시 수동으로 실행하거나 체크한다.

1. **주석 언어 스캔 (필수)**
   - [ ] 파일 내의 모든 주석(`//`, `/* */`)을 확인합니다.
   - [ ] **한글 주석**만 허용됩니다. 영문 주석이 있다면 즉시 번역하여 수정합니다.
   - [ ] 예외: 라이브러리 고유 명사나 기술 용어는 영문 유지가 가능하나, 설명은 한글이어야 합니다.

2. **테스트 검증**
   - [ ] 수정된 로직과 관련된 테스트가 이미 존재하는 경우 실행합니다 (`npm run test` or `npm run b:jest`).

3. **최종 포맷팅**
   - [ ] 코드가 일관된 스타일을 유지하는지 확인합니다.
